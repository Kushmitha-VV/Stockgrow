## 🌟 About the Project

**StockGrow** is a sleek and user-friendly stock market dashboard designed to help users track insights, manage profiles, and visualize stock-related data with a clean and modern UI.

It focuses on simplicity, performance, and a great user experience for investors and learners.

---

## ✨ Features

✔️ Interactive and modern dashboard UI  
✔️ User profile management system  
✔️ Stock data visualization ready design  
✔️ Dark mode support 🌙  
✔️ Notification & alert settings 🔔  
✔️ Fully responsive design 📱 (Mobile + Desktop)  
✔️ Clean and minimal UI/UX  
✔️ Fast performance with Next.js  

---

## 🛠 Tech Stack

- ⚡ Next.js  
- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 💡 JavaScript (ES6+)
